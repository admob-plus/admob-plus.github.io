AdUnitIDOption#

| string

interface IAdRequest {
id?: AdUnitIDOption

childDirected?: boolean
underAgeOfConsent?: boolean
}

id#

Ad unit ID.

Require to be a string or { android: string, ios: string }.

childDirected#

Child-directed setting.

Optional boolean.

Check out the official Android and iOS doc for more info.

testDevices#

Test device IDs.

Optional string[].
